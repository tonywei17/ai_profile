隐私政策

最后更新：2026年3月2日

本隐私政策说明了 Wei Wenxin（以下简称"开发者"、"我们"）在您使用 AI证件照 移动应用程序（以下简称"本应用"）时，如何收集、使用和保护您的信息。

1. 收集的信息

数据类型用途存储方式
您上传的照片AI证件照生成不存储 — 实时处理后立即销毁
生成的照片在应用内展示供您保存仅存储在您的设备上
使用数据管理每日生成次数限制仅存储在设备上（UserDefaults）
订阅状态管理Pro会员由Apple管理（StoreKit 2）
广告标识符展示广告（仅免费用户）由Google AdMob管理

我们不收集：您的姓名、电子邮箱、电话号码、位置信息、通讯录或上述以外的任何个人身份信息。

2. 照片的处理方式

当您上传照片时，将经历以下处理流程：

  1. 在您的设备上进行压缩和缩放（免费用户：512px，Pro用户：1024px）
  2. 通过HTTPS加密传输至我们的后端服务器（托管于 Google Cloud Platform 东京区域）
  3. 转发至 Google Gemini API 进行AI证件照生成
  4. 将生成结果返回给您，原始照片立即销毁

我们不会在服务器上存储、记录或保留您的照片。照片仅在处理期间（通常不超过30秒）存在于服务器内存中。

3. 第三方服务

我们与以下第三方服务共享数据。各服务按照其各自的隐私政策处理数据：

服务提供商共享的数据用途
Gemini APIGoogle LLC您上传的照片 + 文本提示词AI图像生成
AdMobGoogle LLC广告标识符、设备信息展示广告（仅免费用户）
StoreKit 2Apple Inc.订阅交易数据支付处理

使用本应用即表示您知悉并同意您的照片将被发送至 Google Gemini API 用于AI证件照生成。Google按照 Google隐私政策 处理数据。

4. 数据安全

  • 所有数据传输均使用 HTTPS/TLS加密
  • 后端服务器运行于 Google Cloud Run 东京区域（asia-northeast1）
  • 用户照片和个人数据不存储在数据库中
  • API密钥存储在Google Cloud Secret Manager中，不包含在客户端代码中

5. 数据保留

  • 照片：不保留 — 实时处理后立即销毁
  • 使用数据：仅存储在设备上；卸载应用后删除
  • 订阅数据：由Apple管理，遵循Apple的数据保留政策
  • 广告数据：由Google AdMob管理，遵循Google的数据保留政策

6. 您的权利

  • 删除数据：由于我们不在服务器上存储您的照片或个人数据，因此没有需要删除的服务器端数据。卸载应用将删除设备上的所有数据。
  • 退出个性化广告：您可以在设备的"设置 > 隐私与安全 > 跟踪"中进行限制。
  • 撤回同意：您可以随时停止使用本应用。
  • 信息查询：联系我们了解数据处理方式。

7. 儿童隐私

本应用不面向13岁以下（或您所在司法管辖区适用的同意年龄以下）的儿童。我们不会故意收集儿童的个人信息。

8. 地区合规

中国个人信息保护法（PIPL）

面部照片属于敏感个人信息。我们仅在您明确同意（通过上传照片的行为）的情况下处理您的面部照片，且仅用于生成证件照。照片处理后不予保存。

日本个人信息保护法（APPI）

我们将面部照片视为敏感个人信息，仅在获得您的明确同意后处理。

9. 政策变更

我们可能会不时更新本隐私政策。变更将发布在本页面并更新修订日期。变更后继续使用本应用即视为接受修订后的政策。
