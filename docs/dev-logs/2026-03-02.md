## 技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 规格数据结构 | Swift enum + CaseIterable | 类型安全、编译期检查、switch 穷举 |
| 自定义尺寸范围 | 宽 20~60mm、高 20~80mm | 覆盖常见证件照尺寸，避免无意义极端值 |
| 排版打印 DPI | 300 DPI | 便利店打印标准（L判/2L判） |
| iPad 布局 | 单栏 + maxWidth 600 | 简单可靠，避免双栏在不同 iPad 尺寸上的适配问题 |
| 多语言实现 | 手动 switch + LanguageManager | 灵活控制，不依赖系统语言设置 |
| Liquid Glass 阴影 | scrollClipDisabled(true) | iOS 26 内置阴影无法被 .shadow(color: .clear) 覆盖 |

---

## 问题 & 解决方案

| 问题 | 原因 | 解决 |
|------|------|------|
| iPad 双栏布局拥挤 | 内容不足以填充两栏 | 移除双栏，统一单栏 + maxWidth |
| SpecCard 阴影被 ScrollView 截断 | ScrollView 默认 clip | `.scrollClipDisabled(true)` |
| Liquid Glass 阴影无法覆盖 | `.shadow(color: .clear)` 无效 | 改为给阴影更多空间 + scrollClipDisabled |
| Edit tool 匹配重复字符串 | 两处 SpecSelectorView 调用代码完全相同 | 扩大上下文范围使匹配唯一 |
