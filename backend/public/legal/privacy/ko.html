개인정보 처리방침

최종 업데이트: 2026년 3월 2일

본 개인정보 처리방침은 Wei Wenxin(이하 "개발자", "당사")이 제공하는 모바일 애플리케이션 AI 증명사진(이하 "본 앱")을 사용할 때 귀하의 정보를 어떻게 수집, 사용 및 보호하는지 설명합니다.

2. 사진 처리 방식

사진을 업로드하면 다음과 같은 처리가 진행됩니다:

  1. 귀하의 기기에서 압축 및 크기 조정 (무료 사용자: 512px, Pro 사용자: 1024px)
  2. HTTPS를 통해 당사의 백엔드 서버(Google Cloud Platform 도쿄 리전)로 전송
  3. Google Gemini API로 전달하여 AI 증명사진 생성
  4. 생성된 결과를 귀하에게 반환하고 원본 사진은 즉시 삭제

당사의 서버에 사진을 저장, 기록 또는 보관하지 않습니다. 사진은 처리 중(일반적으로 30초 이내)에만 서버 메모리에 존재합니다.

4. 데이터 보안

  • 모든 데이터 전송은 HTTPS/TLS 암호화를 사용합니다
  • 백엔드 서버는 Google Cloud Run 도쿄 리전(asia-northeast1)에서 운영됩니다
  • 사용자의 사진이나 개인 데이터를 데이터베이스에 저장하지 않습니다
  • API 키는 Google Cloud Secret Manager에 보관되며 클라이언트 코드에 포함되지 않습니다

5. 데이터 보존

  • 사진: 보존하지 않음 — 실시간 처리 후 즉시 삭제
  • 이용 데이터: 기기에만 저장; 앱 삭제 시 제거됩니다
  • 구독 데이터: Apple이 관리하며 Apple의 데이터 보존 정책을 따릅니다
  • 광고 데이터: Google AdMob이 관리하며 Google의 데이터 보존 정책을 따릅니다

6. 귀하의 권리

  • 데이터 삭제: 당사 서버에 사진이나 개인 데이터를 저장하지 않으므로 서버 측에서 삭제할 데이터가 없습니다. 앱을 삭제하면 기기 내 모든 데이터가 제거됩니다.
  • 맞춤형 광고 거부: 기기의 "설정 > 개인정보 보호 > 추적"에서 제한할 수 있습니다.
  • 동의 철회: 언제든지 앱 사용을 중단할 수 있습니다.
  • 정보 문의: 데이터 처리 방식에 대해 문의하실 수 있습니다.

7. 아동의 개인정보

본 앱은 13세 미만(또는 귀하의 관할 지역에서 적용되는 동의 연령 미만)의 아동을 대상으로 하지 않습니다. 당사는 의도적으로 아동의 개인정보를 수집하지 않습니다.

8. 개인정보 보호법(PIPA) 준수

안면 사진은 개인정보 보호법에 따라 개인정보로 분류됩니다. 당사는 사진 업로드 행위를 통한 명시적 동의 하에 증명사진 생성 목적으로만 귀하의 사진을 처리합니다. 처리 후 사진을 보관하지 않습니다.

9. 방침 변경

당사는 수시로 본 개인정보 처리방침을 업데이트할 수 있습니다. 변경 사항은 이 페이지에 게시되며 수정 날짜가 업데이트됩니다. 변경 후 앱을 계속 사용하면 수정된 방침에 동의한 것으로 간주됩니다.
