Política de Privacidade

Última atualização: 2 de março de 2026

Esta Política de Privacidade explica como Wei Wenxin ("Desenvolvedor", "nós") coleta, usa e protege suas informações quando você utiliza o aplicativo móvel AI ID Photo ("Aplicativo").

1. Informações que coletamos

NÃO coletamos: nome, endereço de e-mail, número de telefone, dados de localização, contatos ou qualquer outra informação de identificação pessoal além do que está listado acima.

2. Como processamos suas fotos

Quando você envia uma foto, o processo é:

  1. Comprimida e redimensionada no seu dispositivo (512px para usuários gratuitos, 1024px para usuários Pro)
  2. Transmitida via HTTPS para nosso servidor backend (hospedado no Google Cloud Platform, região de Tóquio)
  3. Encaminhada para a API Google Gemini para geração de foto de documento com IA
  4. O resultado é retornado a você e a foto original é descartada imediatamente

Não armazenamos, registramos ou retemos suas fotos em nossos servidores. As fotos existem na memória do servidor apenas durante o breve período de processamento (geralmente menos de 30 segundos).

Ao utilizar este Aplicativo, você concorda que suas fotos serão enviadas para a API Google Gemini para geração de fotos com IA. O Google processa esses dados de acordo com a Política de Privacidade do Google.

4. Segurança dos dados

  • Toda transmissão de dados utiliza criptografia HTTPS/TLS
  • O servidor backend opera no Google Cloud Run na região de Tóquio (asia-northeast1)
  • Nenhuma foto ou dado pessoal do usuário é armazenado em nosso banco de dados
  • Chaves de API são armazenadas no Google Cloud Secret Manager

5. Retenção de dados

  • Fotos: Não retidas — processadas em tempo real e descartadas imediatamente
  • Dados de uso: Apenas no dispositivo; excluídos quando você desinstala o Aplicativo
  • Dados de assinatura: Gerenciados pela Apple
  • Dados de publicidade: Gerenciados pelo Google AdMob

6. Seus direitos

  • Excluir dados: Como não armazenamos fotos ou dados pessoais em nossos servidores, não há dados do lado do servidor para excluir. Desinstalar o Aplicativo remove todos os dados do dispositivo.
  • Recusar anúncios personalizados: Você pode limitar o rastreamento de anúncios em Configurações do dispositivo > Privacidade > Rastreamento.
  • Retirar consentimento: Você pode parar de usar o Aplicativo a qualquer momento.

7. Privacidade infantil

Este Aplicativo não é destinado a crianças menores de 13 anos. Não coletamos intencionalmente informações pessoais de crianças.

8. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. As alterações serão publicadas nesta página com a data de revisão atualizada.
