Kebijakan Privasi

Terakhir diperbarui: 2 Maret 2026

Kebijakan Privasi ini menjelaskan bagaimana Wei Wenxin ("Pengembang", "kami") mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan aplikasi seluler AI ID Photo ("Aplikasi").

Kami TIDAK mengumpulkan: nama, alamat email, nomor telepon, data lokasi, kontak, atau informasi identitas pribadi lainnya selain yang tercantum di atas.

2. Cara Kami Memproses Foto Anda

Ketika Anda mengunggah foto, prosesnya adalah:

  1. Dikompresi dan diubah ukurannya di perangkat Anda (512px untuk pengguna gratis, 1024px untuk pengguna Pro)
  2. Dikirim melalui HTTPS ke server backend kami (dihosting di Google Cloud Platform, wilayah Tokyo)
  3. Diteruskan ke Google Gemini API untuk pembuatan foto identitas berbasis AI
  4. Hasilnya dikembalikan kepada Anda dan foto asli langsung dihapus

Kami tidak menyimpan, mencatat, atau menyimpan foto Anda di server kami. Foto hanya ada di memori server selama periode pemrosesan singkat (biasanya kurang dari 30 detik).

4. Keamanan Data

  • Semua transmisi data menggunakan enkripsi HTTPS/TLS
  • Server backend berjalan di Google Cloud Run wilayah Tokyo (asia-northeast1)
  • Tidak ada foto atau data pribadi pengguna yang disimpan di database kami
  • Kunci API disimpan di Google Cloud Secret Manager

5. Retensi Data

  • Foto: Tidak disimpan — diproses secara real-time dan langsung dihapus
  • Data penggunaan: Hanya di perangkat; dihapus saat Anda menghapus Aplikasi
  • Data langganan: Dikelola oleh Apple
  • Data iklan: Dikelola oleh Google AdMob

6. Hak Anda

  • Hapus data: Karena kami tidak menyimpan foto atau data pribadi di server, tidak ada data sisi server yang perlu dihapus. Menghapus Aplikasi akan menghapus semua data di perangkat.
  • Tolak iklan yang dipersonalisasi: Anda dapat membatasi pelacakan iklan di Pengaturan perangkat > Privasi > Pelacakan.
  • Tarik persetujuan: Anda dapat berhenti menggunakan Aplikasi kapan saja.

7. Privasi Anak

Aplikasi ini tidak ditujukan untuk anak di bawah usia 13 tahun. Kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak.

8. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan dipublikasikan di halaman ini dengan tanggal revisi yang diperbarui.
